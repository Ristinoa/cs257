What is this Webapp for?

This webapp is my first attempt at designing a database-driven web app! This project serves as my CS257 , Software Design, course final project. It consist of an About page (where you find yourself), a Home page Which has... not much aside from a random album viewer. The true prize of this webapp is the Advanced Data Search tab, which is a comprehensive, completely Customizeable search system where you can specify your preferences for any of 13 different unique criteria! Oh, right, the actual dataset itself! This webapp functions using a database constructed from a list of the 5000 top albums of all time (as according to ratemymusic.com). You can view a random album and its metrics on the Homepage (as stated above) or search for albums on the search page!
